# Robust-Medical-Image-Classification-Using-Vision-Transformer
A deep learning framework for pneumonia detection using Vision Transformer (ViT-B/16), DenseNet-121, and Swin Transformer with a soft-voting ensemble. Features MAE self-supervised learning, FGSM/PGD adversarial training, Grad-CAM explainability, and evaluation on the Kaggle Chest X-Ray dataset for robust, generalizable AI.
